이미지의 일부를 이용한 링크걸기

        img src : 이미지를 가져온다.
        map태그 : 특정이미지의 트정부분을 링크걸기 위한 기능을 구현한다.
                    map태그는 페이지의 마지막에 기술한다.
                    usemap을 통해서 이미지를 사용한다.
        area태그 : map태그내에 링크 수만큼 표기한다.
            shape 속성 : rect(사각형), circle(원), poly(다각형)
            coords 속성 : 좌표
